# FoilDex — карточная игра для Telegram Mini App

Коллекционная карточная игра: вскрываешь бустеры, собираешь альбом, дерёшься
с ИИ-соперниками пятёркой карт. Работает полностью офлайн (без своего сервера),
прогресс хранится в Telegram Cloud Storage — синхронизируется между устройствами
игрока автоматически и бесплатно.

Никакого программирования знать не нужно — просто следуй шагам ниже по порядку.

---

## Шаг 1. Залить игру на GitHub Pages (бесплатный хостинг сайта)

1. Зарегистрируйся на [github.com](https://github.com), если ещё нет аккаунта.
2. Нажми **New repository** (зелёная кнопка справа сверху на главной).
3. Назови репозиторий, например `foildex-game`. Сделай его **Public**. Создай.
4. На странице репозитория нажми **Add file → Upload files**.
5. Перетащи туда ВСЕ файлы и папки из этого проекта: `index.html`, `style.css`,
   папку `js` целиком (со всеми файлами внутри). Сохрани структуру — `js` должна
   остаться папкой, а не файлами вперемешку с остальным.
6. Нажми **Commit changes** внизу страницы.
7. Зайди в **Settings** репозитория → слева **Pages**.
8. В разделе **Build and deployment → Source** выбери **Deploy from a branch**,
   ветку `main`, папку `/ (root)`. Сохрани.
9. Подожди 1-2 минуты, обнови страницу — вверху появится зелёная плашка со ссылкой
   вида `https://твой-ник.github.io/foildex-game/`. Это и есть твоя игра.

Проверь: открой эту ссылку в браузере — игра должна открыться и работать
(на этом этапе прогресс сохраняется в самом браузере, это нормально для теста).

---

## Шаг 2. Создать бота в Telegram

1. Открой Telegram, найди **@BotFather**.
2. Отправь `/newbot`.
3. Придумай имя бота (отображаемое, например "FoilDex") и username
   (должен заканчиваться на `bot`, например `foildex_game_bot`).
4. BotFather пришлёт токен — он тебе для этой игры не понадобится, но не
   выкладывай его никому и никуда, это как пароль от бота.

---

## Шаг 3. Подключить сайт как Mini App

1. Всё там же, в чате с **@BotFather**, отправь `/newapp`.
2. Выбери своего бота из списка.
3. Введи название приложения (то, что увидят игроки).
4. Загрузи иконку (картинка 640×360, можно потом заменить).
5. Когда попросит **Web App URL** — вставь ссылку с GitHub Pages из шага 1
   (обязательно с `https://` и слэшем на конце, например
   `https://твой-ник.github.io/foildex-game/`).
6. Готово — BotFather пришлёт ссылку вида `t.me/foildex_game_bot/appname`.

Дополнительно (рекомендую): отправь `/mybots` → выбери бота → **Bot Settings →
Menu Button** → **Configure menu button** → вставь ту же ссылку. Тогда кнопка
"Открыть игру" будет всегда видна прямо в чате с ботом, а не только по ссылке.

---

## Шаг 4. Проверить

Открой ссылку `t.me/твой_бот/appname` с телефона или в самом Telegram —
запустится Mini App, полноэкранно, со своей навигацией (кнопка "назад"
в Telegram будет работать как кнопка "назад" в игре).

## Как получить арт для карт (в том числе для новых)

1. В терминале, в папке `foildex-game`: `python3 tools/generate-art.py`
   (генерирует недостающие картинки в `tools/raw/`, ничего лишнего не трогает).
2. Затем: `bash tools/optimize-art.sh` — сожмёт и разложит их по `assets/cards/`.
3. Залейте `assets/cards/` на GitHub (шаг 1 выше) — готово.

Если позже добавите свои карты в `js/cards.js` — допишите промпт для их `id`
в `tools/card_prompts.json` (можно посмотреть готовый текст-заготовку в
`CARD_ART_PROMPTS.md`) и снова запустите оба скрипта по очереди.

---

## Как потом вносить изменения

Каждый раз, когда захочешь что-то поменять (добавить карты, поправить баланс):
1. Открой файл в GitHub (например `js/cards.js`) → карандаш **Edit** справа сверху.
2. Внеси правки → **Commit changes**.
3. Через минуту GitHub Pages сам обновится, и в Mini App подтянутся изменения
   при следующем открытии (может понадобиться закрыть и заново открыть чат с ботом,
   Telegram кэширует страницу).

## Что уже реализовано в этой версии

- 90 уникальных карт (89 + ваша кастомная n16), 6 стихий, 6 редкостей (Common → Legendary → **Mythic**), 12 визуальных архетипов существ (звери, птицы, драконы, маги, рыцари, нежить, демоны, ведьмы, духи, стражи, змеи, насекомые)
- Многоуровневая pity-система: гарантия Rare+ раз в 10 бустеров, Epic+ раз в
  ~19, Legendary+ раз в ~44, Mythic — жёсткая гарантия раз в ~89 (плюс плавно
  растущие шансы по мере приближения к этим порогам — прогресс виден на
  экране открытия бустера)
- «Везучий бустер» — с шансом ~3.5% редкость всех 5 карт в паке поднимается
  на ступень, с отдельным баннером/анимацией/звуком
- Вскрытие бустеров с анимацией, вспышкой и усиленным звуком/вибро на
  Epic/Legendary/Mythic картах
- Альбом коллекции с фильтром по стихиям, счётчиком дублей
- Полноценный пошаговый бой: мана растёт каждый ход, своя рука, своё поле —
  игрок сам решает, какую карту разыграть и куда атаковать, а не смотрит
  автобой. У ИИ — собственная логика ходов и разменов.
- 20 уровней Арены с 4 боссами, сложность растёт по нарастающей (пул
  редкостей карт ИИ, размер его колоды и здоровье), награда золотом
  масштабируется по уровню (обычные — от 18 до 140, боссы — от 80 до 460)
- Стихийные контр-пары (Вода → Огонь → Природа → Вода, Свет → Тень →
  Молния → Свет) — атака "своей" стихией по слабой стихии соперника даёт
  +30% урона; перед боем показывается баннер матчапа, чтобы подбирать
  колоду осознанно
- Способности карт: щит, яд, лечение, двойной удар, похищение здоровья
- Внутриигровая валюта (кристаллы), ежедневный бонус, стартовые бесплатные бустеры
- Тактильная отдача (вибро) на открытии редких карт — через Telegram Haptics API
- Сохранение через Telegram CloudStorage (с запасным вариантом localStorage
  для тестирования вне Telegram)
- Генератор арта карт одной командой (`tools/generate-art.py`) — бесплатно,
  без API-ключей, с повышенным разрешением для Legendary/Mythic карт
