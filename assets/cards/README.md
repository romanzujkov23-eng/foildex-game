# assets/cards/ — сюда кладутся реальные изображения существ

Формат имени файла = id карты из `js/cards.js`:

    f01.webp   f02.webp   ...   l06.webp

(рядом можно/нужно положить `.jpg` того же имени — запасной
формат, если WebP по какой-то причине не подгрузится).

Список всех id и промптов для генерации — в файле
`CARD_ART_PROMPTS.md` в корне проекта.

Рекомендованный размер: 640×640 (квадрат), см. `tools/optimize-art.sh`
— он сам обрежет и сожмёт любые картинки под этот размер.

Пока файла для карты здесь нет — игра автоматически показывает
старый процедурный SVG-арт, ничего не ломается.
