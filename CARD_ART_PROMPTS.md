# Промпты для арта карт FOILDEX (тёмное фэнтези)

Сгенерируйте по одной картинке на карту в любом генераторе (Midjourney, SDXL/ComfyUI, Leonardo.ai, DALL-E) и сохраните файл как `assets/cards/<id>.webp` (или `.jpg`).

Либо не вручную, а одной командой через `tools/generate-art.py` (см. инструкцию в README и в самом скрипте) — он использует эти же промпты автоматически.

**Карта n16 (Лесная Фея) сюда не входит** — для неё арт уже готов (собственное фото), промпт не нужен.

**Единый стилевой суффикс** (добавляйте в конец каждого промпта, чтобы все карты были в одном визуальном стиле):

> Dark fantasy trading card game illustration, painterly digital art, dramatic chiaroscuro lighting, moody atmospheric background, single creature centered, square composition, high detail, style similar to Magic: The Gathering / Hearthstone concept art, no text, no watermark, no card frame or border --ar 1:1
